\# Online Examination System



\## Overview

This is a web-based Online Examination System built using HTML, CSS, and JavaScript.



\## Features

\- Student Login

\- Admin Login

\- Online MCQ Examination

\- Automatic Score Calculation

\- Result Display

\- Add Questions (Admin)

\- Local Storage Support



\## Technologies Used

\- HTML5

\- CSS3

\- JavaScript



\## How to Run

1\. Download or clone the repository.

2\. Open `index.html` in any web browser.
